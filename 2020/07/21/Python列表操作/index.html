Python

列表操作

遍历列表

for循环

定义一个列表
names = [‘张三’,’李四’,’王五’,’赵六’,’田七’]
zhang_san = names[0]
li_si = names[1]
wang_wu = names[2]
zhao_liu = names[3]
tian_qi = names[4]
print(zhang_san+” “+li_si+” “+wang_wu)

for循环

for name in names:
print(name)

不仅局限为单纯显示，还可以结合别的模块

names2 = [‘anlen’,’bob’]
for name in names2:
print(name.title())

魔法推导式

n = [name for name in names2]

nums = [num**2 for num in range(1,11)]

创建数值列表

range(val1,val2,val3):

参数1：起始值（包含自己）

参数2：终止值（不包含自己）

参数3：步长（间隔数）

range函数
for num in range(1,11):
print(num,end=” “)

输出1-10之间的偶数
for num in range(2,11,2):
print(num,end=” “)

输出1-10之间的奇数
for num in range(1,11,2):
print(num,end=” “)

输出1-10之间的奇数的平方
for num in range(1,11,2):
print(num**2,end=” “)

创建数字列表

numbers = list(range(1,11))
print(numbers)

数字列表的简单统计计算
最大值

max()

最小值

min()

总和

sum()

列表的最大值、最小值、总和
print(max(numbers))
print(min(numbers))
print(sum(numbers))
